Indicator Function

— Stefan Lendl

:CUSTOM_ID: indicator-function

The indicator function or simply function (func) defines the purpose of an indicator within an indicator set.

An indicator contains indicators with the following functions:

  • Confirmation (C1)
  • 2nd Confirmation Indicator (C2)
  • 3rd Confirmation (C3)
  • Volume or Valatility Indicator (V)
  • Baseline (B)
  • Exit (E)
  • Average True Range (ATR)
