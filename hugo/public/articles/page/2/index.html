Analyzing Results

When backtesting indicators or indicator sets there is a significant likelihood of curve fitting which gives good results for the specific backtested period but would not give good results in forward testing. This is also called over-optimization. Filtering Over-Optimization Parameter Optimization describes that if the value of a Continuous Input changes, the result value must change proportionally to the input change. Assumtion: based on Parameter Optimization If the value of a continuous input changes by a single step but the result changes significantly, the results can be considered random and must be filtered.

Combination of Indicators

Implications Indicators may not be compatible :PROPERTIES: :CUSTOM_ID: indicators-may-not-be-compatible :END: Good individual results may be bad in combination with other indicators.
