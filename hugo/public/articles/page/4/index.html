At certain points the indicator gives a signal which indicates to buy/sell at a specific point of time and at a specific price level. Each Indicator has various input parameters which change the behavior of the indicator. Input Parameters

usually referred to as input. Types of Input Parameters An MT5 Indicator may have different types of inputs Integer Float String Boolean (Bool) Enumerations (Enum) Continuous Input An input is continuous if an adjacent input value only slightly changes the function of an indicator. Adjacent input values result in a continuous result space. Typically Integer and Float inputs are continuous. An example would be the the period of an moving average.
