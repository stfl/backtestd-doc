Testing New Indicators

:CUSTOM_ID: testing-new-indicators After a new indicator is defined in the system it shall be tested. Testing means calculating the fitness function (metric) for each input parameter set. The current implementation plan is to process all possible input parameters. This has \(O(n^m)\). \(n\) possible values per input. \(m\) different free inputs. If an indicator has 10k input parameter and it is combined with another indicator of also 10k inputs the resulting search space is \(10k^2 = 100M\)
