Reducing the result set

:CUSTOM_ID: reducing-the-result-set Reducing n n is the number of values of a single input parameter. /The lengths of the input sets are obviously not all the same but for O() notation this does not matter./ The size of n is defined by the number of possible values. The range, describing this has a start, a stop and a step value. If we reduce the move start and stop closer together.
